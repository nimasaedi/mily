# MinRely Deployment Guide

راهنمای نصب و راه‌اندازی پروژه روی هاست (CPanel/DirectAdmin).

## بخش ۱: بک‌اند (Node.js)

1. **آپلود فایل‌ها:** محتویات پوشه `backend` را در روت هاست خود (مثلاً پوشه `api` یا خارج از `public_html` برای امنیت بیشتر) آپلود کنید.
2. **نصب پکیج‌ها:** اگر دسترسی SSH دارید، دستور `npm install` را اجرا کنید. اگر ندارید، پوشه `node_modules` را از لوکال زیپ کرده و آپلود کنید.
3. **تنظیم دیتابیس:** طبق فایل `MIGRATION_AND_SEED_GUIDE.md` دیتابیس را بسازید.
4. **فایل .env:** اطلاعات دیتابیس خود را در فایل `.env` وارد کنید.
5. **اجرا:** اگر هاست شما از Node.js پشتیبانی می‌کند (Setup Node.js App)، فایل `server.js` را به عنوان entry point انتخاب کنید.

## بخش ۲: فرانت‌اند (React)

1. **بیلد گرفتن:** در کامپیوتر خود دستور زیر را اجرا کنید:
   ```bash
   npm run build
   ```
   این دستور یک پوشه `build` یا `dist` ایجاد می‌کند.

2. **آپلود:** محتویات پوشه بیلد شده را داخل `public_html` هاست خود آپلود کنید.

3. **تنظیمات htaccess:** برای اینکه روتینگ React درست کار کند (اگر از BrowserRouter استفاده می‌کردید)، نیاز به کانفیگ داشتید. اما چون ما از `HashRouter` استفاده کردیم، نیاز به تنظیمات خاصی سمت سرور نیست و روی هر هاستی به راحتی بالا می‌آید.

## اتصال فرانت به بک

در فایل `src/services/api.ts` (یا جایی که درخواست‌ها ارسال می‌شوند)، آدرس پایه (Base URL) را به آدرس دامین بک‌اند خود تغییر دهید.
مثال: `https://api.minrely.com` یا `https://minrely.com/api`.
